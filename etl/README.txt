1. Python needs to be installed with all the libraries in requirements.txt installed. 

2. cd to directory where python script is saved. 

3. Save all your downloaded reports into a folder named data.

   root
    |
    |--script.py
    |--data
        |__all the reports 
 
4. Run python <name of script>.py from command line 

5. Then just wait. 

6. A directory named table_files will be created and populated with all the relevant compiled tables.
